How to Use the Console

Setup

  1. Include jQuery <script src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.0/jquery.min.js"></script>
  2. Include the latest Cobold Console code <script src="js/cobold_console-0.1.js"></script>
  3. Include the HTML for the Console 
    <div id="console">
  <div id="consoleOutput">
    Loading...
  </div>
  <div id="consoleInput">
    <form method="get" post="">
      <label for="consoleInputTextArea"></label>
      <textarea id="consoleInputTextArea" autocapitalize="off">
      </textarea>
      <!--- <input type="submit" /> --->
    </form>
  </div>
</div>
  4. Include the CSS styling <link rel="stylesheet" href="css/console_alpha.css" />
  5. Instantiate the Cobold Console 
    $(document).ready(function(){
	window.cc = new ConsoleClass();
});

Functions

Output

  • .add( anything )
  • .addDelay( duration in milliseconds )
  • .addLine( string )
  • .addNewLine()
  • .addText( string )
  • .clear()
  • .refresh()
  • .rush( milliseconds )
  • .setPrompt( string )

Input

  • .acceptOneCharacterInput()
  • .acceptTextEnterInput()
  • .clearInput()
  • .startInput()
  • .stopInput()
  • .submitInput( string )

Special

  • .submitCallback( string )
    Redefine this function so you can send input to your app

Example

	var cc = new ConsoleClass();
	cc.addLine("Hello")
		.addDelay(1000)
		.addLine("World")
		.addDelay(1000);
	cc.submitCallback = function (x) {
		alert(x);
	}
